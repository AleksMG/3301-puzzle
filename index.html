Математическая основа:

• K = 65536 (масштабирующий множитель 2¹⁶)

• Большие простые числа: p и q (50-60 бит)

• N = p × q (100+ бит)

• X = M + i × K, где i ∈ [1, floor((N-255)/K)]

• Шифрование: C = Xe mod N

• Дешифрование: X = Cd mod N, M = X mod K
